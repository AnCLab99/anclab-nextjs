# AnC Lab. — Next.js Website Starter

This starter implements the boutique studio website direction defined for AnC Lab. using:
- Next.js App Router
- TypeScript
- Tailwind CSS

## Included
- Home page with conversion-focused sections
- Work listing page
- Dynamic case-study pages
- Studio page
- Process page
- Contact page
- Reusable components and project data file

## Before running
1. Replace placeholder external image URLs with your own optimized assets.
2. Replace Behance/Instagram/email placeholders in `data/site.ts`.
3. Add your real fonts if you want Satoshi installed locally.
4. Connect the contact form to Formspree, Tally, Resend, or your preferred backend.

## Run locally
```bash
npm install
npm run dev
```

## Suggested next steps
- Add local fonts in `/public/fonts`
- Swap project cover placeholders with your actual thumbnails
- Add SEO metadata per page
- Add CMS or MDX later if you want easier case-study updates
